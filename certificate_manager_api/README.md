# Certificate-Manager

### node - v16.16
### npm - 8.11.0

### Install dependencies 
```bash
npm i
```

### Make Migration
```bash
npx knex migrate:make <migration_name>
```

### Run Migration
```bash
npx knex migrate:latest
```

### Rollback Migration
```bash
npx knex migrate:rollback
```

### Start Backend Server
```bash
npm run start
```
